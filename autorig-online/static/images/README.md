# Images Directory

This directory contains all images and icons for the AutoRig Online website.

## Structure

- `logo/` - Logo files (SVG, PNG, favicon)
- `icons/` - UI icons (SVG format)
- `hero/` - Hero section images
- `gallery/` - Gallery example images
- `process/` - Process/How-it-works images
- `formats/` - File format icons

## Image Generation Prompts

For images that need to be generated (not SVG icons), see the prompts below:

### Open Graph Image (og-image.png)
**Size**: 1200x630px
**Prompt**: Professional Open Graph image for AutoRig Online. Dark background with gradient (indigo to purple). Center: 3D character wireframe or skeleton in T-pose. Text overlay: 'AutoRig Online - Automatic 3D Character Rigging'. Modern, clean design with subtle tech elements. Suitable for social media previews.

### Hero Image (hero/hero-main.jpg)
**Size**: 1920x1080px (desktop), 1200x800px (main)
**Prompt**: Hero image for 3D character rigging service. Abstract 3D wireframe character in T-pose, floating in dark space with indigo/purple gradient lighting. Modern, tech-forward aesthetic. Subtle particles or grid pattern. Professional, clean composition. Space for text overlay on left side.

### Gallery Images
All gallery images should be 1200x900px (4:3 aspect ratio):

1. **fantasy-warrior.jpg**: Fantasy warrior 3D character in T-pose. Armor, weapons, detailed design. Professional game-ready model. Clean background, studio lighting. High quality render.

2. **cyberpunk-character.jpg**: Cyberpunk character 3D model. Mix of mechanical and organic elements. Futuristic design. Clean background, professional render.

3. **animal-companion.jpg**: Quadruped animal companion 3D model. Dog or wolf-like creature in T-pose. Professional game character. Clean background.

4. **mecha-robot.jpg**: Mecha robot 3D character. Mechanical design with visible joints and armor. Professional game-ready model. Clean background.

5. **cartoon-character.jpg**: Stylized cartoon 3D character. Exaggerated proportions, friendly appearance. Game-ready model. Clean background.

6. **animation-showcase-thumb.jpg**: Thumbnail for animation showcase video. Multiple character poses in sequence showing walk cycle or action. Dynamic composition. Indigo/purple gradient background.

### Process Images (process/)
1. **rigging-process.jpg** (1200x800px): Infographic-style illustration showing 3D character rigging process. Before/after comparison: static model on left, rigged skeleton on right. Arrows showing transformation. Clean, modern design with indigo/purple color scheme. Professional, educational appearance.

2. **comparison.jpg** (1200x600px): Split-screen comparison illustration. Left side: traditional manual rigging (complex, time-consuming). Right side: automated AI rigging (fast, efficient). Use icons and simple graphics. Modern infographic style.

3. **technology.jpg** (1000x600px): Abstract visualization of AI/neural network processing 3D geometry. Wireframe mesh being analyzed, nodes and connections. Modern, tech-forward design. Indigo/purple color scheme. Professional, scientific appearance.

### Screenshot (screenshot.png)
**Size**: 1920x1080px
**Prompt**: Screenshot mockup of AutoRig Online website. Show upload interface with 3D character preview. Modern, clean UI design. Browser window frame. Professional appearance.

## Notes

- All SVG icons are ready to use
- Raster images (JPG/PNG) need to be generated using the prompts above
- Images should be optimized for web (WebP format recommended)
- Retina versions (@2x) should be created for high-DPI displays
