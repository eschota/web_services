How AI animal rigging works

1. AI vision detects the body type

AutoRig V2 renders the model from multiple camera angles and detects whether it is humanoid or animal-style.

2. Animal payload uses only rig mode

When the model is an animal, the task uses animal type metadata and focuses the pipeline on rigging first.

3. Animation-ready creature output

The result is designed for creature animation previews, game engines, web showcases and export workflows.
