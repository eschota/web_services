# RenderFin Image Render API (ТЗ / быстрый старт)

Endpoint:
- POST https://renderfin.com/api-render

Назначение:
- Генерация ИЗОБРАЖЕНИЙ (PNG) по текстовому prompt.
- Поддерживается aspect_ratio (адаптация разрешения под соотношение сторон).
- Анимации / видео: пока НЕ поддерживаются (не использовать).

-------------------------
Формат запроса (JSON)
-------------------------
Минимально достаточно:

{
  "prompt": "a photorealistic portrait of a woman, studio light",
  "aspect_ratio": 1.777
}

Рекомендуемые поля:

{
  "prompt": "описание того, что нужно сгенерировать",
  "negative_prompt": "что НЕ должно быть в кадре (опционально)",
  "aspect_ratio": 1.0,          // ОБЯЗАТЕЛЬНО если нужен другой формат. width/height
  "user_name": "any_name"       // опционально (для группировки результатов)
}

aspect_ratio примеры:
- 1.0    = 1:1 (квадрат)
- 1.777  = 16:9 (горизонтально)
- 0.5625 = 9:16 (вертикально)

-------------------------
Разрешение результата
-------------------------
- По умолчанию (если aspect_ratio не указан или = 0): 1280x1024
- Если aspect_ratio указан:
  - разрешение подбирается автоматически
  - максимальная сторона = 1280 px
  - пропорции сохраняются

-------------------------
Ответ
-------------------------
Успех:

{
  "output_url": "https://renderfin.com/render/<user_name>/<task_id>.png"
}

Это прямая ссылка на готовый PNG.

-------------------------
Ошибки
-------------------------
400 BadRequest с текстом ошибки (например: пустой prompt или некорректный aspect_ratio).
